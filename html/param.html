After selecting all the parameters please press the Calculate Button.

  • Disease: The default values, and allowed ranges, for the basic reproduction number (R0), the generation time (Tg) and the population are shown in the table below. To introduce a new disease, with larger ranges for R0 and Tg select the Build Your Own Disease option.
    Disease R0 Tg (Days) Population
    Seasonal Influenza 1.3 (1.2-1.4) 2.6 (2.0-3.0) 10,000 (10,000-106)
    Pandemic Influenza 1.5 (1.3-1.7) 3.0 (2.0-3.0) 10,000 (10,000-106)
    SARS 2.7 (2.2-3.7) 10. (6.0-11.0) 10,000 (10,000-106)
    Ebola 1.7 (1.2-2.1) 12. (2.0-21.0) 10,000 (10,000-106)
    Build Your Own 1.4 (1.1-3.0) 3.0 (2.0-15.0) 10,000 (10,000-106)

  • Basic Reproduction Number (R0) - Default values and allowed range for R0 are shown above. This parameter can be modified using the sliding bar.
  • Generation Time (Tg) - Default values and allowed range for Tg are shown above. This parameter can be modified using the sliding bar.
  • Population - The Population is set at 10,000. To select a different value use the drop down menu.
  • Susceptible Contact Rate - This parameter (denoted as p in the plots) describes a reduction (from an initial value of one) in the contact rate of the susceptible population due to a change in bevavior. This parameter can be modified by using the sliding bar.
  • Infectious Contact Rate - This parameter (denoted as q in the plots) describes a reduction (from an initial value of one) in the contact rate of the infectious population due to a change in bevavior. This parameter can be modified by using the sliding bar.
  • Start Day of Behavior Change - For the time-dependent behavior change model this is the start day of behavior change (reduction in contact rate). It is measured with respect to day zero of the disease.
  • Time for Behavior Change to Take Full Effect - For the time-dependent behavior change model this is half the time that it takes for behavior change to take full effect.
